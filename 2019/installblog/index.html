Hugo Installation and Blog Creation

一、Install Hugo

1. Download hugo binary file

2. Hugo binary file doesn’t need to be installed,just add paths to system environment variable PATH.(My Computer-Attribute-Advanced System Settings-Environment Variable)Then select PATH,click Edit,and finally copy location of hugo binary file to PATH.

(PS. this is windows system.if mac,you can use package management tool—brew.)

二、Generate site

1. command line—hugo new site [site’s name].(eg. hugo new site myblog,default installation on C disk.)

2. command line—hugo new site [site’s name].(eg. hugo new site E:/mybloy)

三、Set theme

1. Find theme of hugo

2. Not the root directory under the themes directory,and clone theme(git clone https://github.com/liuzc/LeaveIt.git)

四、Local startup

1. Under root directory to implement command line—hugo server -t leaveit --buildDrafts(-t represent theme,and leaveit reprement the name of the theme installed.)

2. Under root directory to create artical to implement command line—hugo new [path+name.md()markdown].(eg. hugo new post/index.md)

五、Deploy to Git

1. Create a repository

2. Hugo relate git,and generate a public file. command line—hugo --theme=leaveit --baseUrl="https://xiaolinfan.github.io/" --buildDrafts

3. Just upload public file to git,and here’s the command line

git init
git add .
git commit -m "notes"
git remote add origin https://github.com/xiaolinfan/xiaolinfan.github.io.git
git push -u origin master

六、Perfect blog

1. Site setting in the config.toml

2. When add categories and tags,notice that there is a space behind the “:“,otherwise the error will be reported.

3. After adding blog,you can deploy from step 六,or implement deploy.sh(continued)
